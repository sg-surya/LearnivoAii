# **App Name**: Sahayak AI Lite

## Core Features:

- Role-Based Authentication: Separate sign-up/login flows for 'Teachers' to protect the app and its models.
- Dashboard: A central hub for teachers, providing access to AI tools and generated assets. This will serve as the main navigation point within the app.
- My Library: Allow teachers to manually add and digitize their textbooks and educational content.
- Lesson Planner: AI-powered tool to generate weekly lesson plans based on topic, grade, and objectives, powered by a Genkit flow, which acts as a tool to determine suitable resources.
- Hyper-Local Content Generation: Generate stories, examples, and content tailored to the teacher's location and language.
- Instant Knowledge Base: Provide age-appropriate answers to student questions in the local language.
- Parent Communication: Draft professional emails to parents in the local language.

## Style Guidelines:

- Primary color: Indigo (#4B0082) for a professional, trustworthy feel.
- Background color: Light gray (#F0F0F0) for a clean and simple design.
- Accent color: Gold (#FFD700) for highlighting important elements and CTAs.
- Headline Font: 'Space Grotesk' sans-serif for bold and modern headlines.
- Body Font: 'Inter' sans-serif for readable and accessible content.
- Use consistent and clear icons from 'lucide-react' for a modern look.
- Responsive grid-based layout with a persistent sidebar for easy navigation.
